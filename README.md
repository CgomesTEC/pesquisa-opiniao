# pesquisa-opiniao
Projeto em Python para pesquisa de opinião utilizando estrutura de repetição e decisão.
## 📌 Funcionalidades
- Entrada de dados do usuário
- Classificação de respostas
- Uso de estruturas de decisão (if, elif, else)

## ▶️ Como executar

Execute o arquivo `pesquisa-opiniao.py` em um ambiente Python  
ou utilize o Google Colab para rodar o arquivo `.ipynb`.
